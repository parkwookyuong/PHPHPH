# PHPHPH

## 설치 방법
1. Python 3.8 이상이 필요합니다.
2. 프로젝트 클론:
   ```bash
   git clone https://github.com/yourusername/PHPHPH.git
   cd PHPHPH

### 사용 방법
1. 설치 후, FastAPI 서버를 실행하려면 다음을 실행하세요:
phpyphishing

2. 브라우저에서 확인:
